# Network Traffic Analysis Lab

This repository documents my practice analyzing network traffic for cybersecurity purposes.

## Topics covered

- DNS queries
- TCP handshake
- ICMP traffic
- ARP protocol
- suspicious traffic patterns

## Tools

- Wireshark
- tcpdump
- Linux command line

## Goal

The goal of this project is to build practical skills in network traffic analysis used in cybersecurity and SOC environments.
